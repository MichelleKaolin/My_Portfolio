## 💼 Featured Projects

### NEXDERM™ — Predictive Beauty Intelligence
A predictive AI platform that creates a digital twin of each user's skin, simulating how it will look in 5, 10, and 20 years based on lifestyle, environment, and profession — and recommending the exact L'Oréal products to optimize that future. Developed for the **L'Oréal Brandstorm 2026** global competition.

### Robotic Arm with Arduino
A functional robotic arm with programmed movement control. Arduino executes commands in C++, while a C# interface sends real-time instructions — integrating hardware and software with precision.

### GOTTA — Environmental Monitoring
A web application simulating a drone company focused on wildfire prevention. The system analyzes environmental data, generates automatic reports, and issues risk alerts.
